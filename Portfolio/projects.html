67Zip Trojan Ransomware

Visit Project

67Zip is a Windows file compression tool with built-in context menu integration for zipping and unzipping files. While it functions like a standard compression utility, it also secretly operates as a ransomware trojan. Whenever a user compresses or extracts a file, the malware encrypts their documents in the background. This project was created for a cybersecurity class to demonstrate how seemingly harmless applications can be weaponized.
