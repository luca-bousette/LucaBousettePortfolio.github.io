# Luca Bousette's Portfolio Site

A personal portfolio website created to showcase my projects, technical skills, and professional experience. This site serves as a central hub for potential employers and collaborators to learn about my work and get in touch.

## About This Project

This portfolio website was built as part of my Job Preparation course at John Abbott College. It demonstrates my ability to create responsive, professional web applications using core web technologies.

## Purpose

The main goals of this portfolio are to:
- Present my background, education, and technical skills in a professional manner
- Showcase my best programming and game development projects
- Provide easy access to my GitHub repositories and project demos
- Offer multiple ways for potential employers or collaborators to contact me

## Features

- **Responsive Design** - Fully functional on desktop, tablet, and mobile devices with a hamburger menu for smaller screens
- **Modern Dark Theme** - Professional aesthetic with smooth animations and hover effects
- **Projects Page** - Dedicated section highlighting my work with direct links to GitHub repositories
- **Interactive Skills Display** - Visual grid showcasing my technical proficiencies across languages, frameworks, and tools
- **Contact Cards** - Easy-to-use contact section with email, GitHub, and LinkedIn links
- **Downloadable Resume** - Quick access to my resume directly from the navigation bar
